# Tic Tac Toe – C Programming Project

## 📌 Description
This project implements a **console-based Tic Tac Toe game** using the C
programming language.  
It allows two players to play the game turn by turn, validates user input,
updates the game board, and determines the winner or draw condition.

The project focuses on **game logic implementation, modular programming,
and structured C code design**, which help build strong programming fundamentals.

---

## 🧠 Concepts Used
- Arrays
- Functions
- Conditional statements
- Loops
- Modular programming
- User input validation

---

## 📁 File Structure

```
tic-tac-toe/
├── main.c         # Program entry point and game control
├── playgame.c     # Game logic and board handling
├── game_header.h  # Function declarations and macros
└── README.md
```

---

## ▶️ Build and Run

Compile the program using GCC:

```bash
gcc *.c -o tic_tac_toe
```

Run the executable:

```bash
./tic_tac_toe
```
